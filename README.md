# Tailwind CSS 文档

[在线中文文档](http://tailwindcss.cool)

Tailwind CSS 使用[Next.js](https://nextjs.org/)做文档，你可以自己本地运行项目：

1. Clone项目

    ```sh
    git clone https://github.com/jellykaiser/tailwindcss.cool.git
    ```

2. 进入文件夹

    ```sh
    cd tailwindcss.cool
    ```

3. 安装依赖

    ```sh
    yarn
    ```

4. 启动本地服务

    ```sh
    yarn dev
    ```

你可以使用`yarn export`命令将文档编译成静态资源并导出。导出的文档在项目根目录的`out`文件夹.
